## Technical Decisions & Rationale

### Wrapping ResponsiveContainer for Accessibility
**Problem:** Recharts' `ResponsiveContainer` doesn't accept `role` and `aria-label` attributes in TypeScript (type safety).
**Solution:** Wrap ResponsiveContainer in a `<div role="img" aria-label="...">` element while keeping the chart component unstyled.
**Benefit:** Maintains semantic HTML accessibility while respecting component type definitions.

### Plain-Language Volatility Descriptors
Converted mathematical volatility coefficient (CV = std dev / mean) into plain Indonesian descriptions so users understand forecast accuracy without needing statistical knowledge:
- Consistent spending → better predictions → lower forecast volatility
- Volatile spending → wider prediction ranges → higher forecast uncertainty

### Token-Driven Color System
All color references now flow through centralized helpers:
```tsx
// Before: Hardcoded hex or inline vars
stroke="var(--color-content-tertiary)"    // ❌
fill="#8884d8"                             // ❌

// After: Semantic token helpers
stroke={colorVar('content-tertiary')}      // ✅
fill={colorVar('brand-primary')}           // ✅
```
Single source of truth: `frontend/src/styles/theme.css`
