### Overview.tsx — Spending Pattern Insight

**State Management:**
- `chartInsight: string | null` — Stores Claude Haiku response
- `insightLoading: boolean` — Tracks API call status

**API Call (useEffect):**
```typescript
const generateInsight = async () => {
  if (!transactions || transactions.length === 0) return;
  setInsightLoading(true);
  try {
    const response = await fetch('/api/ask-assistant', {
      method: 'POST',
      body: JSON.stringify({
        question: 'Analisis ringkas pola pengeluaran saya bulan ini. Fokus pada kategori terbesar, tren pengeluaran, dan saran penghematan praktis.',
        transactions,
        month,
        year,
      }),
    });
    if (response.ok) {
      const result = await response.json();
      setChartInsight(result.response);
    }
  } catch (error) {
    console.error('Failed to generate chart insight:', error);
  } finally {
    setInsightLoading(false);
  }
};
```

**Triggers On:** `[transactions, month, year]` — Regenerates when transactions or filter changes

**Display:**
- ChartInsight placed after "Grafik Pengeluaran" collapsible section
- Icon: "📊"
- Shows loading skeleton while API is generating
- Shows fallback text if API fails or returns error

**Example Output (in Indonesian):**
> "Pengeluaran Anda turun 12% bulan ini. Kategori terbesar masih Makanan (35%). Saran: Pertahankan momentum tabungan dan pertimbangkan meal prep untuk efisiensi."

## API Integration

**Endpoint:** `/api/ask-assistant` (Vercel serverless function at `frontend/api/ask-assistant.js`)

**Input Payload:**
```json
{
  "question": "User's contextual question in Indonesian",
  "transactions": [Transaction[], ...],
  "month": 5,
  "year": 2026
}
```

**Processing:**
1. Aggregates transaction data (total income, total expense, category breakdown)
2. Builds context string in Indonesian with financial summary
3. Sends prompt + context to Claude Haiku (`claude-haiku-4-5-20251001`)
4. Returns JSON with `response` field containing insight text (max 500 tokens)

**Response Format:**
```json
{
  "response": "Insight text in Indonesian (2-3 sentences)"
}
```

**Error Handling:**
- Network error → log to console, show fallback text
- API timeout (>5s) → gracefully degrade, show fallback
- API error (500) → return fallback text
- No breaking failures → chart always renders, insight is bonus
