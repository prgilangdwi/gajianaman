## 3. ARCHITECTURE NOTES
- **PolaWaktu Calendar Logic**: Daily breakdown uses Date API (getDate, getMonth, getFullYear) for accurate month filtering. useMemo dependency list: [transactions, calendarMonth.month, calendarMonth.year]
- **DetailKategori Period Toggle**: Weekly calculation derives week boundaries from dayOfWeek offset; monthly calculation uses Intl.DateTimeFormat for locale-aware labels. Data recalculation triggered automatically on viewPeriod state change
- **Component Reusability**: Both pages follow established patterns from Overview, Tren, Pengeluaran pages — Motion animations, theme tokens (textColorVar, bgColorVar, borderColorVar), Card/Button components from shadcn/ui, useReducedMotion accessibility hook
- **URL Routing**: DetailKategori expects route parameter `:category` from react-router v7; category name is URI-encoded to handle spaces
- **Error Handling**: Both components use ScreenStates pattern (LoadingState, ErrorState, EmptyState) for consistent UX
- **Mobile Responsiveness**: Calendar uses grid grid-cols-7 for responsive layout; DetailKategori uses flex wrapping and min-w-0 for text truncation on 375px viewport
