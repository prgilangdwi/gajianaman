## 3. ARCHITECTURAL DECISIONS

### Zero Dead Code Enforcement
- **Approach:** Delete everything not in Master Roadmap Section 5.1 (15 essential screens)
- **Why:** Dead code inflates bundle size, increases maintenance burden, violates AETHER anti-bloat model
- **Authority:** User's explicit ruling on "definitive, authoritative" page retention list

### Backward Compatibility Maintenance
- **Approach:** Keep essential redirects (/overview → /home/overview, /pengeluaran → /spend/spending) but remove redirects for deleted pages
- **Why:** Users with bookmarks to old URLs still transition gracefully to current structure
- **Trade-off:** Reduced redirect clutter while maintaining UX continuity
