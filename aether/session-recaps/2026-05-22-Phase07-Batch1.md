## Technical Implementation Notes

### SwipeAction.tsx
- **Gesture**: Touch-based left/right swipe with spring physics (stiffness: 300, damping: 30)
- **Accessibility**: Keyboard fallback menu with long-press affordance; aria-label on all actions
- **Colors**: Semantic (sentiment-negative for delete, orange for flag)
- **Constraints**: Max translate 120px; swipe threshold 80px

### BottomSheet.tsx
- **Snap Points**: Configurable; defaults to [30vh, 100vh] for progressive disclosure
- **Gesture**: Swipe-to-dismiss with 100px threshold; backdrop click to close
- **Animation**: `bottomSheetEnter` preset (300ms, exit ease)
- **A11y**: Focus trap, role="dialog", aria-modal="true", Escape key dismiss
- **Visual**: Rounded top (radius-3xl), handle indicator for affordance

### PullToRefresh.tsx
- **Momentum Detection**: Checks scroll position at touch start; enables only when at top
- **Resistance Easing**: `sqrt(distance) * 3` for natural deceleration feeling
- **Indicator**: Rotating icon + status text (Indonesian: "Lepaskan untuk segarkan" / "Menyegarkan...")
- **Performance**: Passive event listeners (no jank on 60fps scroll)
- **Visual**: Color changes (slate → green) when threshold reached; spinner animation
