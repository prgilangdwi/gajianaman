## Technical Decisions & Rationale

### Why Stacked Area Chart for Income vs Expense?
The stacked area chart in Tren.tsx provides better visual storytelling than side-by-side lines:
- **Before:** Two independent line traces (LineChart) made comparison harder
- **After:** Stacked areas with 0.6 opacity show the relationship between income and expenses clearly
- **Trade-off:** Stacked areas are harder to read exact values (mitigated by Tooltip)

### CSS Variable System
All color references now flow through token helpers:
```tsx
// Before: Hardcoded hex or Tailwind classes
stroke="#10b981"          // ❌
className="text-red-600"  // ❌

// After: CSS variables via helpers
stroke={colorVar('sentiment-positive')}    // ✅
className={textColorVar('content-primary')} // ✅
```
This enables single-source-of-truth color management via `frontend/src/styles/theme.css`.

### Accessibility Approach
- All SVG charts wrapped in ResponsiveContainer with `role="img"`
- Descriptive `aria-label` attributes include chart type + time context
- Follows WCAG 3 guideline for complex data visualizations
- Screen readers can navigate and understand chart purpose without visual cues

## Known Limitations & Future Work

### Limitations (By Design)
1. **Category Trend Chart** — Fixed to 3 months in phase requirements; time toggle in main hero chart doesn't affect category breakdown
   - **Rationale:** Category trend is historically less relevant beyond 3 months; keeping it static reduces cognitive load
   - **Future:** Could extend if user feedback indicates need for longer periods

2. **Tooltip Colors** — Tooltip text colors still inherited from Recharts' internal color mapping (not user-configurable)
   - **Rationale:** Recharts Tooltip API provides colors automatically; changing would require custom implementation
   - **Future:** Custom Tooltip component could use colorVar helpers if needed

### Out-of-Scope for Next Batch
- Export charts to PDF (handled in Laporan via existing `exportLaporanToPDF`)
- Drill-down interactions from charts to transaction details
