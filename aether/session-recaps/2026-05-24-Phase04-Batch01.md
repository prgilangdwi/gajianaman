## 3. ARCHITECTURE NOTES

- **Component Structure:**
  - Gajian.tsx conditionally renders GajianWizard when `showWizard` state is true
  - GajianWizard is a self-contained form component with onBack callback
  - Both follow established patterns from Overview, Pengeluaran pages
  
- **State Management:**
  - Step state is local to GajianWizard (no global context needed yet)
  - Form data (`GajianInput`) is typed and validated at each step
  - useCallback optimization prevents unnecessary re-renders of child inputs
  
- **Validation Pattern:**
  - Step-specific validation functions: `canProceedStep1()`, `canProceedStep2()`, `canProceedStep3()`
  - `isStepValid()` calls appropriate validator based on current step
  - Next button disabled until step validation passes
  
- **Motion Design:**
  - Container animations: page enter (fade in)
  - Step transitions: AnimatePresence with variant switching (fade + slide)
  - Element animations: staggered fade-up on initial load
  - Progress bar: animated width based on step
  - Button hovers: conditional scale animation
  - All animations bypass when prefers-reduced-motion is detected
  
- **Theme Integration:**
  - Uses existing utility functions: formatRupiah(), cn(), bgColorVar(), textColorVar(), borderColorVar()
  - All colors via CSS variables (no hardcoded colors)
  - Consistent with existing design system (theme.css)
  - Respects dark/light mode toggle
  
- **Accessibility:**
  - Proper Label + Input associations (htmlFor/id)
  - Semantic HTML (button, input, label)
  - prefers-reduced-motion respected (all animations disabled)
  - Focus-visible CSS built into Button component
  - Step indicator text updates to guide users
  - ARIA attributes on radio buttons (via Radix foundation)
  
- **Mobile Responsiveness:**
  - max-w-2xl container ensures readability on all viewport sizes
  - Grid layouts use responsive columns (grid-cols-1 md:grid-cols-3)
  - Input fields use full width on mobile, appropriate width on desktop
  - Button text and icons scale appropriately
  - No horizontal scroll on 375px viewport
